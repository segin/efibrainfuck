# EFI Brainfuck 

## What is EFI Brainfuck? 

EFI Brainfuck is, as the name implies, a Brainfuck interpreter for Extensible
Firmware Interface (EFI) and it's successor, Universal Extensible Firmware 
Interface (UEFI).
